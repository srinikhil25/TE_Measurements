# Assets Folder

Store static assets like images, logos, and icons here.

## Logo Placement

Place your logo image file here:
- **Recommended name**: `logo.png` or `logo.jpg`
- **Supported formats**: PNG, JPG, JPEG, SVG
- **Recommended size**: 200x200px to 400x400px for best quality

## Current Structure

```
assets/
├── logo.png          # Main logo (place your logo here)
└── README.md         # This file
```
